Teachers Space

Please, take your time and explore the system. Two major concepts are intervined in the systems mechanics and those are 'courses' and 'questions'. Following is a brief explanation on the functionality regarding both of them.

Courses

Before you can start creating and issuing questions, make sure that you are enrolled on the course for which you would like to create a question for. If the course does not exists, feel free to create one, as long as you keep in mind that the course number should be unique! You will be automatically enrolled on all the courses you create.


         
Create a New Course                  Enroll on a Course

Questions

After being enrolled on a course, fell free to create as many questions as you like for said course. Please, make use of our detailed question creator form to get the most advantage of the system! Finally, you have to issue the question and give the generated question code to your students so they can answer it. This can be easily achieved from the 'Browse Questions' menu, where you can also use the 'tag' system to search for existing questions and observe the statistics for a specific question in a simple, yet elegant way.
